Time Capsule
A web app where you can write a message and lock it until a future date. When the date arrives, the capsule unlocks and you (or someone else) can finally read it.

What it does
You write a message, pick a date, and the app hides it until then. You can send it to yourself or to another user. Once it unlocks, you can choose to make it public so others can see it and react to it. Capsules are organized by categories like dreams, career, or personal.
